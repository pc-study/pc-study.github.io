# 💬 2020-01-27

**【单选题】关系型数据库中Table表是什么含义？**

- [ ] A、展示相同数据结构的数据的集合
- [ ] B、存储不同数据结构的数据的集合
- [ ] C、展示不同数据结构的数据的集合
- [x] D、存储相同数据结构的数据的集合

**<font color='green'>正确答案：D</font>**

**<font color='red'>解析：</font>** 在关系型数据库中，表（TABLE）是数据库中用来存储数据的对象，是具有相同结构的数据的集合，是整个数据库系统的基础。

# 💬 2020-01-28

**【单选题】SQL命令统计数量？**

- [ ] A、COUNT * from Table_Name
- [ ] B、SELECT COUNTALL() from Table_Name
- [x] C、SELECT COUNT(*) from Table_Name
- [ ] D、COUNTALL() from Table_Name

**<font color='green'>正确答案：C</font>**

**<font color='red'>解析：</font>** `SQL COUNT()` 函数返回匹配指定条件的行数。

**SQL COUNT(*) 语法**
```sql
SELECT COUNT(*) FROM table_name
```

# 💬 2020-01-29

**【单选题】InnoDB存储引擎最低支持什么级别锁？**

- [ ] A、字段锁
- [x] B、行锁
- [ ] C、数据库锁
- [ ] D、表锁

**<font color='green'>正确答案：B</font>**

**<font color='red'>解析：</font>** InnoDB和MyIsam是Mysql中两种主要的存储引擎，InnoDB最低支持行级锁，MyIsam最低支持表级锁，表级锁性能低。使用场景，一般默认都是InnoDB，如果表比较小，读取操作较多，插入和更新操作少，不需要考虑事务，可以考虑MyIsam存储引擎。

# 💬 2020-01-30

**【单选题】SQL命令查询不重复记录？**

- [x] A、SELECT Distinct id from Table_Name
- [ ] B、SELECT NUM from Table_Name
- [ ] C、SELECT ALL from Table_Name
- [ ] D、SELECT Difference * from Table_Name

**<font color='green'>正确答案：A</font>**

**<font color='red'>解析：</font>** 关键词 `DISTINCT` 用于返回唯一不同的值。在表中，可能会包含重复值。这并不成问题，不过，有时您也许希望仅仅列出不同（distinct）的值。

**语法：**
```sql
SELECT DISTINCT 列名称 FROM 表名称
```

# 💬 2020-02-01

**【单选题】MySQL事务中原子性指的是？**

- [x] A、事务中的状态不能分离
- [ ] B、事务中的SQL命令不可分割执行
- [ ] C、事务中的表不可分割
- [ ] D、事务中的数据不可分割

**<font color='green'>正确答案：A</font>**

**<font color='red'>解析：</font>** 原子性是指事务包含的所有操作要么全部成功，要么全部失败回滚，换言之，一次操作是不可分割的，要么全部成功，要么全部失败。比如我们的转账操作，不允许出款方成功，收款方失败这种情况，要么都成功，要么多失败，不可能出现中间状态。

# 💬 2020-02-02

**【单选题】什么是1NF？**

- [ ] A、表的每一列是可分割的基本数据项
- [x] B、表的每一列都是不可分割的基本数据项
- [ ] C、表的某一列是不可分割的基本数据项
- [ ] D、表的某一列是可分割的基本数据项

**<font color='green'>正确答案：B</font>**

**<font color='red'>解析：</font>** 数据表中的每一列（每个字段）必须是不可拆分的最小单元，也就是确保每一列的原子性。

# 💬 2020-02-03

**【单选题】SQL如何创建一个表？**

- [ ] A、CREATE DataTABLE table1 (c1 datatype,c2 datatyp);
- [ ] B、CREATE MyTABLE table1 (c1 datatype,c2 datatyp);
- [ ] C、CREATE SQLTABLE table1 (c1 datatype,c2 datatyp);
- [x] D、CREATE TABLE table1 (c1 datatype,c2 datatyp);

**<font color='green'>正确答案：D</font>**

**<font color='red'>解析：</font>** CREATE TABLE 语句用于创建数据库中的表。

**SQL CREATE TABLE 语法：**
```sql
CREATE TABLE 表名称
(
列名称1 数据类型,
列名称2 数据类型,
列名称3 数据类型,
....
)
```

**【单选题】RDBMS是什么？**

- [x] A、Relational Database Management Systems
- [ ] B、Relational Database Manipulate Systems
- [ ] C、Rela Database Management Systems
- [ ] D、Relational Database Management Speed

**<font color='green'>正确答案：A</font>**

**<font color='red'>解析：</font>** RDBMS 是 Relational Database Management System 的缩写，中文译为“关系数据库管理系统”，关系数据库管理系统（RDBMS）是一种基于 E. F. Codd 发明的关系模型的数据库管理系统（DBMS）。
