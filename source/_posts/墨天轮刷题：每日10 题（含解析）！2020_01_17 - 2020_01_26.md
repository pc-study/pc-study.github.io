# 💬 2020-01-17

**【单选题】什么是3NF？**

- [x] A、满足2NF，表中不包含已在其它表中已包含的非主关键字信息
- [ ] B、满足1NF，表中包含已在其它表中已包含的非主关键字信息
- [ ] C、满足1NF，表中不包含已在其它表中已包含的非主关键字信息
- [ ] D、满足2NF，表中包含已在其它表中已包含的非主关键字信息

**<font color='green'>正确答案：A</font>**

**<font color='red'>解析：</font>** 范式依次类推，因此需满足 2NF，并且第三范式（3NF）不包含已在其它关系已包含的非主关键字信息。

# 💬 2020-01-18

**【单选题】DBMS是什么含义？**

- [ ] A、Database Management Speed
- [x] B、Database Management Systems
- [ ] C、Data Management Systems
- [ ] D、Database Manipulate Systems

**<font color='green'>正确答案：B</font>**

**<font color='red'>解析：</font>** 数据库管理系统(Database Management System)是一种操纵和管理数据库的大型软件，用于建立、使用和维护数据库，简称DBMS。

# 💬 2020-01-19

**【单选题】SQL命令根据ID删除一条记录？**

- [ ] A、DELETE ALL from Table_Name WHERE ID =值
- [x] B、DELETE from Table_Name WHERE ID =值
- [ ] C、DELETE * from Table_Name WHERE ID =值
- [ ] D、DELETE ONE * from Table_Name WHERE ID =值

**<font color='green'>正确答案：B</font>**

**<font color='red'>解析：</font>** DELETE 语句用于删除表中的行。

**语法：**
```sql
DELETE FROM 表名称 WHERE 列名称 = 值
```
# 💬 2020-01-20

**【单选题】MySQL事务中一致性指的是？**

- [ ] A、事务中的表记录字段数量一致
- [ ] B、事务中的SQL命令保持一致
- [x] C、事务修改前后的数据总体保证一致
- [ ] D、事务中的表记录数量一致

**<font color='green'>正确答案：C</font>**

**<font color='red'>解析：</font>** 为了保证事务的正确执行，维护数据库的完整性，事务必须具有以下特性：原子性、一致性、隔离性和持久性。其中一致性是指事务的隔离执行(没有并发其他事务)保持数据库的一致性。

# 💬 2020-01-21

**【单选题】MySQL中触发器的作用是？**

- [x] A、触发器是针对数据库的操作（增删改）发生时被调用的特殊存储过程
- [ ] B、触发器用来复制数据库
- [ ] C、触发器用来新增数据
- [ ] D、触发器用来保存日志

**<font color='green'>正确答案：A</font>**

**<font color='red'>解析：</font>** 触发器是一种特殊的存储过程，它在插入，删除或修改特定表中的数据时触发执行，它比数据库本身标准的功能有更精细和更复杂的数据控制能力。

# 💬 2020-01-25

**【单选题】MySQL事务中隔离性指的是？**

- [x] A、事务之间相互隔离
- [ ] B、事务的连接之间是相互隔离的
- [ ] C、事务中的SQL命令是相互隔离执行的
- [ ] D、事务中的数据是相互隔离的


**<font color='green'>正确答案：A</font>**

**<font color='red'>解析：</font>** 事务之间相互隔离！即同一时间，只允许一个事务请求同一数据，不同的事务之间彼此没有任何干扰。比如A正在从一张银行卡中取钱，在A取钱的过程结束前，B不能向这张卡转账。

**【单选题】什么是索引Index？**

- [ ] A、SQL数据库里的目录工具
- [x] B、SQL数据库用来加速数据查询的特殊的数据结构
- [ ] C、SQL数据库里的表管理工具
- [ ] D、SQL数据库里的查询工具

**<font color='green'>正确答案：B</font>**

**<font color='red'>解析：</font>** 索引是一种特殊的数据库结构，由数据表中的一列或多列组合而成，可以用来快速查询数据表中有某一特定值的记录。
