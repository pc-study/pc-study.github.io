昨天写了一篇 [19.30 先别用了，有 BUG！](https://mp.weixin.qq.com/s/_rkU33udfXpy5eq26wwi0w)，有些朋友比较焦虑，因为刚刚升级了 19.30，慌了！

先别急，官方已经提供了 one-off 补丁进行修复：

![](https://oss-emcsprod-public.modb.pro/image/editor/20260126-2015708808937562112_395407.png)

昨天紧急下线修复版上线了，并且提供了 README 文档。

我已经第一时间下载了：

![](https://oss-emcsprod-public.modb.pro/image/editor/20260126-2015710738069938176_395407.png)

one-off 补丁的安装方式也十分简单：
```bash
## 以 19.30 环境为例，上传补丁包到 /soft 目录下
unzip -q /soft/p6880880_190000_Linux-x86-64.zip -d $ORACLE_HOME
opatch version
cd /soft
unzip -q /soft/p38854064_1930000DBRU_Linux-x86-64.zip
cd /soft/38854064
opatch apply -silent
opatch lspatches
```
按照以上步骤安装修复补丁即可。
