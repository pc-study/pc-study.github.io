# 前言
自从 Oracle MOS 更新版本之后，备受吐槽，各种问题纷至沓来，我也写了两篇文章聊了聊这个事：
- [Oracle MOS 重磅升级？嗯，什么东西！](https://mp.weixin.qq.com/s/YYXx9uOSv1Pb8di15VWLdA)
- [听说 Oracle MOS 又放大招！](https://mp.weixin.qq.com/s/cmIwZKVrC11dolEHGHSYUA)

今天登陆 MOS，发现 12.18 号更新了一篇文章：

![](https://oss-emcsprod-public.modb.pro/image/editor/20251222-2002978644822679552_395407.png)

**文章的大概意思就是**：关于用户之前吐槽 MOS 的大多数问题，我们都已经进行修复或者在修复中了。

![](https://oss-emcsprod-public.modb.pro/image/editor/20251222-2002979108955971584_395407.png)

## 已解决的问题
1. **文档搜索功能修复**：之前用户无法通过旧版文档编号（如 472449.1）搜索知识文章，且部分文档不显示，现已提升搜索能力并修复链接重定向。
2. **旧版收藏夹恢复**：用户可通过 **`Account → Preferences → Previous Favorites`** 查看，**仅迁移以“.1”结尾的文章收藏**，原文件夹内的收藏现以平铺列表展示，已删除文章将无法访问。
3. **文章打印功能修复**：之前从门户直接打印时格式不理想，现已修复。浏览器打印功能得到增强，文章标题可完整显示而不被截断。
4. **文章内大型表格显示修复**：之前包含多列表格的文章显示不正常，现已完成格式优化并实施了进一步改进。

## 改进中
1. **外部链接重定向问题**：从外部网站（如谷歌）或旧书签访问知识文章的链接可能无法正常跳转，相关链接处理功能仍在持续优化。
2. **大屏幕文章显示优化**：目前文章以固定宽度显示，未充分利用大屏空间，正在更新以适应不同屏幕尺寸。

# 总结
目前看起来对我的帮助不大，但是这个态度还是值得点赞，希望优化的越来越好！
