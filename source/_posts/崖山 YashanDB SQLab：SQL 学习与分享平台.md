# 前言
记得之前不少朋友问我有没有一些在线的 SQL 实验网站，用来学习一些 SQL 基础语法和学习测试。今天发现一款崖山数据库的在线 SQL 学习与分享平台：
>**SQLab**：[https://sqlab.yashandb.com](https://sqlab.yashandb.com)

![](https://oss-emcsprod-public.modb.pro/image/editor/20241217-1868904924709404672_395407.png)

🎈 轻松上手，无需安装：
- 直接在线体验 YashanDB，无需安装任何软件，即可编写和运行 SQL 和 PL/SQL 脚本。
- 简洁直观的图形化界面，集成教材和运行环境，一键执行样例 SQL，让学习变得轻松便捷。

📘 丰富教程，学习无忧：
- 提供 SQLhandbook、深入浅出 YashanDB 等内置教材，从基础到进阶，全方位覆盖。
- 内置样例数据库，让你在实践中快速提升数据库操作技能。

📒 笔记广场，共享智慧：
- 支持在教材基础上直接生成笔记或新建笔记，记录你的学习心得，巩固知识。
- 公开笔记功能，让你的笔记成为他人学习的参考，与全国 SQL 学习者交流经验，共同成长。

我试用了一下，感觉十分便捷，基本满足大部分的使用需求，比如：
1. 在线执行 SQL
2. 一键美化 SQL
3. 历史 SQL 回看
4. 记录 SQL 笔记
5. AI 智能问答助手

等等功能，大家可以自行探索！

# SQLab 初体验
